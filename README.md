# sunrise-alarm

Sunrise alarm clock

Hardware:

* Adafruit Trinket M0
* Adafruit Audio FX Sound Board
* DS3231 precision real-time clock (on Adafruit breakout board)
* TI PCF8574N i2c I/O expander
* Sparkfun i2c 7-segment LED display
* addressable RGB LEDs

Software/Libraries:
* Arduino IDE
* AceButton
* ioAbstraction
* Kelvin2RGB
* RTCLib
* Wire/SPI
